# 판결문 기반 네이버 블로그 글 생성기

PDF 판결문을 입력하면 OpenAI API를 호출해 변호사 홍보용 네이버 블로그 글 초안을 생성하는 로컬 Windows 앱입니다.

## 기능

- PDF 판결문 텍스트 추출
- 한국어 판결문 PDF 처리를 위해 PyMuPDF 우선 사용, pypdf 보조 사용
- 스캔본 PDF로 판단되면 OpenAI Vision OCR 자동 사용
- OCR은 기본 3페이지 병렬 처리와 `detail=auto` 설정으로 속도와 비용을 조정
- `gpt-5.5`처럼 `temperature`를 지원하지 않는 모델은 자동으로 해당 파라미터를 제거하고 재시도
- API 전송 전 기본 민감정보 마스킹
- API Key 로컬 저장 및 삭제
- 지정 프롬프트 기반 네이버 블로그 글 생성
- 작성 분야, 타깃 독자, 지역 키워드, 변호사/법무법인명, 상담 CTA 입력 지원
- 옵션 선택 시 본문 삽입용 이미지 PNG 생성
- 제목, 본문, 요약, 태그, 썸네일 문구, 게시 전 체크리스트 생성
- 광고 리스크 표현 자동 경고
- 결과 TXT 저장

## 실행

전제:

- Windows 10 이상
- Python 3.10 이상
- `python --version` 명령이 정상 동작해야 함
- Python 설치 시 `tcl/tk and IDLE` 옵션 포함
- OpenAI API Key
- PDF 판결문

Windows에서 개발 실행:

```bat
cd C:\blog_writer
run.bat
```

exe 빌드:

```bat
cd C:\blog_writer
build_exe.bat
```

빌드 결과는 `dist\BlogWriter.exe`에 생성됩니다.

## 사용 방법

1. OpenAI API Key를 입력합니다.
2. `API Key 저장`을 누릅니다. 다음 실행부터 자동으로 불러옵니다.
3. 판결문 PDF를 선택합니다.
4. 일반 텍스트 추출이 어려운 스캔본이면 OpenAI OCR이 자동 실행됩니다.
5. 추출된 텍스트와 마스킹 결과를 확인합니다.
6. 작성 분야, 타깃 독자, 지역 키워드, 변호사/법무법인명, 상담 CTA를 입력합니다. 비워두면 판결문 기반으로 추론하거나 기본값을 사용합니다.
7. 이미지도 생성하려면 `본문 이미지 생성`을 체크하고 이미지 수를 선택합니다.
8. `블로그 글 생성`을 누릅니다.
9. 결과를 검수한 뒤 저장합니다.

## 주의

- API Key는 `%APPDATA%\BlogWriter\config.json`에 저장됩니다. 앱 화면의 `삭제` 버튼으로 지울 수 있습니다.
- 스캔본 PDF는 OpenAI OCR을 사용하므로 API 비용이 추가로 발생합니다.
- 이미지 생성 옵션을 켜면 이미지 수만큼 추가 비용이 발생합니다. 기본값은 비용을 낮추기 위해 2장, `quality=low`, `1024x1024`입니다.
- 스캔본 OCR을 하려면 PDF 선택 전에 OpenAI API Key를 먼저 입력해야 합니다.
- OCR은 최대 30페이지까지 처리합니다.
- `/UniKS-UTF16-H` 같은 PDF 인코딩 경고가 보이면 `run.bat` 또는 `build_exe.bat`를 다시 실행해 최신 의존성을 설치하세요.
- 생성 결과는 초안입니다. 게시 전 변호사가 사실관계, 개인정보, 광고 규정 위반 가능성을 반드시 검수해야 합니다.
- 유사 사건이라도 구체적인 사실관계와 증거에 따라 결과는 달라질 수 있습니다.
- API Key는 앱 메모리에서만 사용하며, 별도 저장 기능은 넣지 않았습니다.

기본 모델은 `gpt-4.1-mini`입니다. 앱 화면에서 모델명을 수정할 수 있습니다.
